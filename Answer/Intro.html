Data Visualisation

D3 is very powerful JavaScript library written and developed by Mike Bostock for visualising data. If you look on Mikes web page you can get an idea of the impressive things which can be done with this library. This tutorial is a short introduction to D3 and a collection of resources where you can find out more information.

3 little circles

This is a small exercise has been taken from Mikes 3 little circles tutorial and will act as a quick introduction to JavaScript and D3. You can follow the tutorial on blocks.org and edit this file to see the results. Also remember you can run things in the browser console on Chrome or Firefox, Right click and click "inspect elements" to pull up the developers console.
